nmap Commands

  • nmap -sS [target]

    > Perform a stealth scan
  • nmap -O [target] - Detect operating system
  • nmap -p- [target] - Scan all 65535 ports
  • nmap -A [target] - Perform an aggressive scan
  • nmap -T4 [target] - Speed up the scan
  • nmap --script [script_name] [target] - Run a specific script
