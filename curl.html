curl Commands

  • $ curl https://example.com

    > retrieve the HTML content of the specified URL

  • $ curl -X GET https://api.example.com/resource

    > send a GET request.

  • $ curl -X POST -d "key1=value1&key2=value2" https://api.example.com/resource

    > send a POST request with data.

  • $ curl -F 'bigUploadFile=@rce.php' 'http://example.com/bigUpload.php?action=post-unsupported'

    > -F flag specifies a form field (bigUploadFile) with the file content.
